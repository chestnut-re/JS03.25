WEEK Monday

Git与GitHub

Git与GitHub的概念、关系

  • Git：代码管理工具

  • GitHub：项目代码托管平台

  • 通过Git（Git命令、图形化界面），将本地代码上传到 GitHub-远程仓库

用Git关联GitHub

  1. 首先在GitHub创建一个远程仓库

  2. 复制远程仓库地址 https://github.com/*******（用户名）/201902JS（远程仓库名）.git

  3. 找到本地指定位置

    1. 右键打开Git bash here 命令窗口

    2. 将远程仓库下载到本地

      • git config – global user. 本地Name、本地email

    3. git clone 远程仓库地址

      • git 本地仓库里都有一个默认隐藏的.git文件夹勿动

      • 查看本地仓库与远程仓库关联的语法git remote -v

    4. git add

      • git add .将本地仓库内代码从工作区全部上传到暂存区

      • git add 指定具体代码的名称将本地仓库指定代码上传

        1. git status 查看当前仓库内文件的状态

      • 出现红色代码表示文件还存于工作区

      • 出现绿色代码表示文件已上传到暂存区

        1. git commit -m”备注信息” 将暂存区的文件上传到历史区

      • 备注信息必须得有

        1. git push origin master 将历史区文件上传到远程仓库

      • origin （公式性）远程代码仓库地址

      • master（公式性）本地所在位置

        push后 所关联的GitHub帐号登录

获取老师远程仓库资料

  • git clone 别人远程仓库地址

  • 拉取最新代码 pull拉取 git pull origin master

  • 如果是直接 git clone 下来的仓库 已经默认建立关联了 不需要再建立

  • 将本地已有文件夹 变成Git仓库 和 远程指定仓库建立关联

  • git init 把已有本地文件夹 初始化成git仓库

建立关联

  • git remote add origin 远程仓库地址

Git常用命令

创建文件夹

mkdir 文件夹名

递归创建文件夹

mkdir -p 11/22

创建文件名

touch 文件名
touch index.js

查看文件内容

cat 文件名

切换目录 cd

cd 目录名 进入某个目录
cd .. 切换到上级目录

查看当前所在路径

pwd

查看当前目录下所有文件

ls

查看所有文件包含隐藏文件

ls -al

rm -rf 强制删除文件
